| Company | Pricing | Features |
|---------|---------|----------|
| **Botnoi AI** | ฿5,000-50,000/เดือน | LINE/Facebook/WhatsApp, Voicebot, AI Agent Builder |
| **Wisesoft** | ฿10,000-100,000/เดือน | LINE API expert, Custom dev, RPA |
| **eBOT** | ฿3,000-30,000/เดือน | LINE chatbot specialist, E-commerce |
| **SiamAI** | ฿5,000-25,000/เดือน | Thai language AI, Custom LLM |

## Market Insights

- **LINE OA ในไทย:** ~2-3 ล้านบัญชี
- **SME ไทย:** ~3 ล้านราย
- **ราคาเฉลี่ย:** ฿3,000-30,000/เดือน

## Pain Points (ปัญหาที่ลูกค้าเจอ)

1. **ค่าใช้จ่ายสูง** - Custom development 50,000+ บาท
2. **เอกสาร LINE API ยาก** - Dev เข้าใจยาก
3. **ภาษาไทย AI ยังไม่แม่น** - Chatbot ทั่วไปพูดไทยไม่ดี
4. **Integrate ยาก** - ต่อกับ POS/ERP ลำบาก
5. **ขาด tech expertise** - SME ไม่มีคนดูแล
