# MeowChat - Customer Features & Strengths

## 🔧 What Customers Can Configure

### AI Agent Settings
| Setting | Description | Example |
|---------|-------------|---------|
| **Agent Name** | ชื่อ AI ที่ลูกค้าจะเห็น | "แมวส้ม", "พี่หมาย", "สมุนไพร" |
| **Agent Personality** | บุคลิก AI | เป็นกันเอง, เป็นมืออาชีพ, ตลก |
| **Greeting Message** | ข้อความต้อนรับ | "สวัสดีค่ะ! มีอะไรให้ช่วย?" |
| **Response Tone** | น้ำเสียง | อ่อนหวาน, เป็นทางการ, กึ่งทางการ |
| **Working Hours** | เวลาทำงาน | 9:00-18:00 หรือ 24/7 |

### Business Information
| Setting | Description |
|---------|-------------|
| **Shop Name** | ชื่อร้าน |
| **Business Type** | ประเภทธุรกิจ (ร้านขายของ, คลินิก, ร้านอาหาร) |
| **Products/Services** | สินค้าและบริการ |
| **Pricing** | ราคา |
| **Working Hours** | เวลาเปิด-ปิด |
| **Contact Info** | เบอร์, ที่อยู่, LINE ID |

### Auto-Reply Rules
