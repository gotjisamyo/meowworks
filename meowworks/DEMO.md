# AI Agent Demo - Web Version

## Quick Demo (ไม่ต้องติดตั้งอะไร)

Demo นี้แสดงให้เห็นว่า AI Agent ทำงานยังไง

### Demo Scenario: ร้านขายเสื้อผ้าออนไลน์

**ลูกค้าถาม:** "มีเสื้อสีน้ำเงินไซส์ L ไหม?"

---

### 🤖 Without AI Agent (ปกติ)

```
ลูกค้า: มีเสื้อสีน้ำเงินไซส์ L ไหม?
พนักงาน: [ต้องหยิบมือถือขึ้นมาตอบ]
พนักงาน: รอสักครู่นะคะ เช็คสต็อกให้
(5 นาทีต่อมา...)
พนักงาน: มีค่ะ สีน้ำเงิน ไซส์ L มีอยู่ 1 ตัว ราคา 350 บาทค่ะ
```

**เวลา:** 5+ นาที/ลูกค้า  
**ค่าใช้จ่าย:** 15,000 บาท/เดือน

---

### ✨ With AI Agent (ของเรา)

```
ลูกค้า: มีเสื้อสีน้ำเงินไซส์ L ไหม?

🤖 AI Agent:
├── ตอบทันที (0.5 วินาที)
├── เช็คสต็อกอัตโนมัติ
└── ตอบ: "มีค่ะ! สีน้ำเงิน ไซส์ L มี 1 ตัว ราคา 350 บาท 
         สนใจสั่งซื้อเลยไหมคะ? 😊"
```

**เวลา:** 0.5 วินาที/ลูกค้า  
**ค่าใช้จ่าย:** 999 บาท/เดือน
