# MeowChat - MeowChat - Documentation

## Product Overview
MeowChat คือ SaaS สำหรับ SME ไทย ช่วยให้ธุรกิจใช้ AI แทนพนักงาน

## Features
- AI Chat Agent
- Order Management
- Multi-channel (LINE, Web, Telegram)
- Analytics Dashboard

## Quick Start
1. สมัคร account
2. เชื่อมต่อ LINE หรือ Widget
3. ตั้งค่า AI Agent
4. เริ่มใช้งาน

## API Documentation
- POST /api/chat - ส่งข้อความ
- GET /api/orders - ดู orders
- POST /api/agents - สร้าง agent
