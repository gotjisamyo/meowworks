# Project: AI Agent Manager for SME Thailand

**เริ่ม:** 14 มีนาคม 2026  
**CEO:** ก็อต  
**VP:** แอนนา (AI)  

## ไอเดีย
ทำ AI Agent Manager สำหรับ SME ไทย - ช่วยร้านค้าใช้ AI แทนพนักงาน

## คู่แข่ง
- Automation Anywhere (enterprise)
- Zapier (แพงเกินไปสำหรับ SME ไทย)
- ไม่มี local ในไทย

## Target
SME ไทยที่ใช้ LINE OA

## ความคืบหน้า (14 มี.ค. 2026)

### ✅ ทำเสร็จแล้ว:
- Landing Page สวยๆ
- Dashboard พร้อม Stats
- Auth System (Register/Login)
- Multi-shop (หลายร้าน)
- Billing (3 แพลน)
- Settings Page
- AI Chat (MiniMax API ตอบจริง)
- Web Chat Widget Demo

### 📁 โครงสร้าง:
```
meowworks/
├── backend/       # API Server (port 3001)
├── frontend/      # Dashboard (port 3000)
├── widget/        # Chat Widget Demo
├── landing/       # Landing Page
├── FEATURES.md    # ฟีเจอร์ลูกค้า
├── USP.md         # จุดเด่น
└── SUMMARY.md    # สรุป
```

### 🌐 URLs:
- Dashboard: http://localhost:3000
- Settings: http://localhost:3000/settings
- Landing: http://localhost:8081
- Widget Demo: http://localhost:8082/demo.html

### ⏳ ยังต้องทำ:
- Deploy ขึ้น Cloud
- LINE SDK Integration

### 💡 ชื่อโปรเจกต์: MeowWorks (เปลี่ยนจาก AI Agent Manager)

### ⚠️ สถานะจริง:
- AI ใช้ข้อมูลตัวอย่าง (hardcoded) - ยังไม่ได้เชื่อม Database จริง
- ต้องสร้าง Products Table และ Products API
- ต้องเพิ่ม Products Page ใน Dashboard
